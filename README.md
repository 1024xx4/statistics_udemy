# 分布（distribution)
どの値にどれくらい Data が存在するかを表したもの

## Histogram
- １つの連続変数の分布を表すのに使用する。
- 各棒の区間に感覚がない
- 横軸は Data の**区間**で切られている
- bin を使ってその区間の度数（frequency）を表す。

## 棒 Group（bar chart）
- 棒を使った Graph の総称
- Category 変数の分布を表すのに使用する。
- 各棒の区間には間隔がある。
- 横軸は Category 変数のとりうる値
- 棒を使ってその値の度数（frequency）を表す
